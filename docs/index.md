# Mapa do Racismo API

Bem-vindo à documentação oficial da API do Mapa do Racismo, uma plataforma para mapeamento e documentação de ocorrências de racismo no Brasil.

## Visão Geral

O Mapa do Racismo é uma API RESTful que permite:

- Registrar e consultar tipos de racismo
- Mapear localizações onde ocorreram casos de racismo
- Registrar respostas e avaliações sobre ocorrências
- Visualizar dados estatísticos sobre casos reportados

## Tecnologias

Esta API foi construída usando:

- Node.js e Express
- Prisma ORM
- PostgreSQL
- Jest para testes
